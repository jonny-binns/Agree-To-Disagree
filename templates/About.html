About Agree to Disagree

Data Use

We collect no personal or identifiying data on our users. The only data that we do collect is all of the debates, responses and votes that occur. If you would like to see the data that has been collected by Agree to Disagree click the button on the main menu.

About the Project

The idea of this project was to create a tool for generating datasets of argument data that can be used in research. This dataset can be downloaded from the main menu. Feel free to use this dataset in whatever projects you would like, data is stored in SADFace formatted json files (the project can be found here). Agree to Disagree is open source and the GitHub repo can be found here, feel free to run it on a personal server and collect your own dataset. The website is a python flask based app so it can be ran like any other flask app.
